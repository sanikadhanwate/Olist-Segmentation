# 📊 Customer Segmentation & Behavioral Clustering  
### Olist E-Commerce Dataset

End-to-end customer segmentation pipeline integrating 100K+ transactional records to identify behavioral clusters and support revenue-focused marketing strategy.

---

## 📌 Project Overview

This project develops a data-driven customer segmentation framework using transactional e-commerce data from Olist (Brazil). The objective is to identify distinct customer groups based on purchasing behavior and translate insights into actionable business strategies.

The analysis integrates multiple relational datasets and applies unsupervised machine learning techniques to uncover customer heterogeneity.

---

## 🎯 Business Objectives

- Identify high-value and repeat customers  
- Detect at-risk and low-engagement users  
- Enable targeted marketing and retention strategy  
- Improve revenue optimization through behavioral segmentation  

---

## 📂 Dataset

- 100K+ orders  
- Multiple relational tables (customers, orders, payments, reviews, geolocation)  
- 1M+ geolocation records  

Data was merged and transformed to create customer-level behavioral profiles.

---

## 🛠 Feature Engineering

Engineered customer-level features including:

- Recency (days since last purchase)  
- Frequency (number of purchases)  
- Monetary value (total spend)  
- Payment behavior  
- Geographic attributes  

Data was standardized and prepared for clustering analysis.

---

## 🤖 Modeling Approach

Applied unsupervised learning techniques:

- K-Means Clustering  
- DBSCAN  
- HDBSCAN  
- Principal Component Analysis (PCA)  

Model performance was evaluated using:

- Silhouette Score (Best: 0.43)  
- Cluster interpretability  
- Business coherence  

---

## 📈 Key Findings

Identified distinct customer segments:

- High-value loyal customers  
- Price-sensitive repeat buyers  
- Low-engagement / one-time purchasers  
- At-risk customers  

Significant behavioral differences were observed across spending patterns, purchase frequency, and recency metrics.
